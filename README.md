# ai-agent   #
simple agent simple task .
requirements are :
FUNCTIONAL API KEY
ADK(AGENT DEVELOPMENT KIT).
PYTHON VERSION 3 or java or GO and abo .
Create vitrual environment, python -m venv venv
Activate virtual environment,.\venv\Scripts\Activate.ps1
for window cmd   use  .venv\Scripts\activate.bat
for powershell use   .venv\Scripts\Activate.ps1
for linux use   source .venv/bin/activate
Install pip install adk..pip install google-adk
Add your api to env file
Write your code to  agent.py if only you are using  python language.
Learn about import and from 
To run use  either web or terminal 
For web we write ..adk web --port 8000    
 for command line we write ..adk run [name of folder].
